# pip命令

## 

1. pip install <...>
	- 安装第三方库	
	- -U 更新库
	- --upgrade 同上
	- -r xx.txt 从文件中批量安装

2. pip uninstall <...>
	- 卸载第三方库
	- -r xx.txt 从文件中批量卸载

3. pip download <...>
	- 下载但不安装

4. pip show <...>
	- 列出某个库的详细信息

5. pip search <...>
	- 根据关键字在名称和介绍中搜索第三方库

6. pip install <文件名>
	- 安装本地二进制安装包

7. pip list
	- 列出已安装的第三方库
	- -o 可升级的库

8. pip freeze
	- 以标准格式列出已安装库
	- >  xx.txt 导出到文件中
## 

1. python -m pip install --upgrade pip
	 - 升级pip
